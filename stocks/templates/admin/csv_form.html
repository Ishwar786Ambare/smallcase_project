Import Stocks from CSV/Excel

Instructions:

  • Upload a CSV or Excel file containing stock symbols
  • The file should have a column named 'symbol' or 'Symbol' (or use the first column)
  • Stock details (name, price) will be automatically fetched from Yahoo Finance
  • Select the exchange (NSE or BSE) - the appropriate suffix will be added automatically

Example CSV format:

symbol
RELIANCE
TCS
HDFCBANK
INFY
Cancel

Notes:

  • If a stock already exists, its information will be updated
  • Stocks that cannot be fetched from Yahoo Finance will be skipped
  • This process may take a few minutes for large files
  • Make sure you have an active internet connection
